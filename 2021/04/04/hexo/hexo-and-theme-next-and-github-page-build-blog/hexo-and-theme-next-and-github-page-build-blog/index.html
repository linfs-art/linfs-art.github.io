首页

（1）首页可以自定义（即可以在/home/目录下创建index.md文件，在里面自定义要显示的内容）。默认情况（/home/目录下没有index.md）下点击首页会显示所有博客（默认全文显示）。

（2）首页显示博客摘要

方法1：

直接在文章中需要截断的地方加入：
